HTLM

  • Hypertext Markup Language (HTML) is used to give the content structure.
  • It is made up of "elements" wich work together to provide content and the basic framework on the web.
  • This elements also divide and describre the content to developers
  • The head element contains information about the webpage.
  • The body elemente represents the visible content shown to the user.

CSS

  • Cascading Style Sheets (CSS) give the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using CSS, you can control exactly how HTML elements will look in the browser.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates anew branch and switches to it.
  • Git is great for tracking changes locally on your own computer.

JavaScript

  • JavaScript (JS) is used to handle logic and allows users to interact with websites.
  • A variable is a named container that allows us to store fata in our code.
  • Control flow is the order in wich a computer executes code in a script.
  • An array is a single variable that is used to hold a group of data.
  • A function is a set of instructions that tells the computer how to perform a certain task.
